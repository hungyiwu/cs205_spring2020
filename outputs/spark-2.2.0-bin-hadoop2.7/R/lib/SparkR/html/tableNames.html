Table Names

Description

Returns the names of tables in the given database as an array.

Usage

## Default S3 method:
tableNames(databaseName = NULL)

Arguments

databaseName

(optional) name of the database

Value

a list of table names

Note

tableNames since 1.4.0

Examples

## Not run: 
sparkR.session()
tableNames("hive")

## End(Not run)
